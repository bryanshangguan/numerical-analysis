# CS323 HW3 (Numerical Analysis)

This directory contains a complete solution package for HW3:

- Problem 1: Proof of polynomial representation via Lagrange basis and partition of unity.
- Problem 2: Newton interpolating polynomial implementation, max error evaluation, and plotting.
- Problem 3: Analytical calculation of the maximum error bound for quadratic interpolation.

## Run

From the repository root (ensure `matplotlib` is installed for the plot):

```bash
python hw3/main.py
```
