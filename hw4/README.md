# CS323 HW4 (Numerical Analysis)

This directory contains a complete solution package for HW4:

- Problem 1: operation counts for dot products, matrix-vector products, and matrix-matrix products.
- Problem 2: LU factorization without pivoting and solution of the given linear system.
- Problem 3: proofs for the elimination-matrix identities used in LU factorization.
- Problem 4: counterexample showing `max_(i,j) |a_(ij)|` is not a matrix norm.

## Run

From the repository root:

```bash
python hw4/main.py
```
